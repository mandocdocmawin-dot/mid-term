# Coffee Supply Management System

## Database Table Fields (`coffee_supplies`)

- `uuid`
- `coffee_id`
- `name`
- `category` (Arabica, Robusta, Liberica)
- `quantity`
- `unit_of_measure` (kg, lbs, oz)
- `price`
- `created_at`
- `updated_at`
